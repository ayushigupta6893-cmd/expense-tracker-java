# 💰 Expense Tracker Application

## 📌 Description
This is a Java-based web application developed using JSP and Servlets that allows users to manage and track daily expenses efficiently. The application performs CRUD operations and stores data in a MySQL database using Hibernate ORM framework.

## 🚀 Features
- Add new expenses
- View expense list
- Delete expenses
- Perform CRUD operations
- Database integration using Hibernate

## 🛠 Technologies Used
- Java
- JSP
- Servlets
- Hibernate
- MySQL
- Maven
